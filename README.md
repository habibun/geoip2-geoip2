# geoip2-geoip2
Implements GeoIP2 PHP API in symfony

| Branch   | GeoIP2 Version | Symfony Version | PHP Version     |
|----------|----------------|-----------------|-----------------|
| [2.x][1] | `2.x`          | `4.4.x`         | `7.4` or higher |


Read [GeoIP2 PHP API Docs](https://github.com/maxmind/GeoIP2-php)
